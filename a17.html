Knowledge Tree

Fundamentals are finite.

At any given moment, there are only so many principles. Things can be broken down in granular detail, but the returns of doing so diminish beyond a certain point. Eventually, we hit a wall.

What does this mean for medicine?

Will we run out of fundamentals, and then fall back into non-productive drudgery?

I don't think so.

Fields are connected. We can take the lessons from one, and apply them to another. Better yet, we can pull together distant fields to gain access to a broader, more unique set of problems.

There is no upper bound.

The more we work, the more principles we learn, the more opportunities we notice, the more problems we can tackle, and the better the future can become.
